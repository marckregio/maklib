# makunatlib [![](https://jitpack.io/v/marckregio/makunatlib.svg)](https://jitpack.io/#marckregio/makunatlib)
## Android Library

	**Abstract Classes**
	* API.java
	* AsyncRequest.java
	* BaseActivity.java
	* Broadcast.java
	* Files.java
	* Preferences.java
	
	**Util Classes**
	* Permissions.java
	* NetworkCheck.java
	* DownloadFiles.java
	* DownloadImage.java
	* Unzip.java
	

	ADD THIS TO PROJECT LEVEL BUILD.GRADLE
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
	
	ADD THIS TO APP LEVEL BUILD.GRADLE
	dependencies {
		compile 'com.github.marckregio:makunatlib:[version]'
	}
